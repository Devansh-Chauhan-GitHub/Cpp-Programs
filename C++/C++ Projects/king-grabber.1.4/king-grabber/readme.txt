King Grabber is a WordPress post plugin which helps you improve your site content with our rich grabber.

== Description ==

King Grabber is a WordPress plugin which helps you for improve your site content with rich features grabbing. We offer more than 10 Actively host comic and community service like TMDB, IMDB
To use this plugin you need first to register your application on our site here for free <a href="https://kinggrabber.com/">https://kinggrabber.com/</a>
You can find our official documentation at <a href="https://kinggrabber.com/docs">https://kinggrabber.com/docs</a> and our Privacy and Policy on our official site at <a href="https://kinggrabber.com/members">https://kinggrabber.com/privacy</a>.

Features in King Grabber include:

* Single Grab Chapter on 10+ Comic Host
* Multi Grabber
* Addons Grabber like IMDB and more

== Installation ==

Upload the King Grabber plugin to your site, Activate it.
That's it. You're done!
